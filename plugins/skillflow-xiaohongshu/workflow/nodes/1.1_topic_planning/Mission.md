## 职责

围绕单篇小红书笔记完成选题策划，并在缺少账号定位时初始化 workflow 层共享账号定位文件。账号定位是该 workflow 的长期上下文，后续 task 默认复用同一份定位。

## 输入与参考

- 本次 task 的主题、素材、产品信息、目标或用户补充要求。
- 当前节点 workflow 层 `Context/account-positioning.md`。如果该文件不存在或内容不足，本节点负责先采集并补齐。

## 执行步骤

// 建立或复用账号长期定位

检查 `Context/account-positioning.md` 是否存在且包含账号领域、目标人群、账号人设、内容目标、表达语气、禁区和视觉偏好。若缺失，需要基于 task 需求采集必要信息并写入该文件；已有有效定位时直接复用。若本次 task 明确要求调整定位，可以更新该文件，并在产出中说明更新依据。

// 将本次素材转成可执行选题

基于账号定位和本次输入，明确本篇笔记的目标读者、内容目标、选题角度、核心痛点、点击钩子、互动点和适合的小红书表达方向。

// 为后续文案与配图提供结构

输出本篇笔记的内容大纲，说明建议采用的表达类型，例如教程、观点、测评、种草、清单、读书笔记、产品功能拆解或经验分享。

## 产出

- `outputs/topic-plan.md`：本篇笔记选题策划文档。
- 必要时创建或更新 workflow 层 `Context/account-positioning.md`。

## 边界

本节点不生成完整正文，不生成图片，不制作预览网页。
